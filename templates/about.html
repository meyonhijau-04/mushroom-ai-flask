Tentang Aplikasi

MushroomAI adalah aplikasi web berbasis kecerdasan buatan yang menggunakan algoritma Random Forest untuk mengklasifikasikan jamur sebagai dapat dimakan (Edible) atau beracun (Poisonous) berdasarkan gambar yang diunggah.

Aplikasi ini dikembangkan sebagai tugas praktikum mata kuliah Kecerdasan Buatan menggunakan Python dan Flask sebagai backend, serta Bootstrap 5 untuk antarmuka pengguna.

Peringatan Penting

Aplikasi ini dikembangkan semata-mata untuk keperluan akademik dan edukasi. Hasil prediksi yang dihasilkan tidak boleh dijadikan satu-satunya acuan dalam menentukan keamanan jamur untuk dikonsumsi. Selalu konsultasikan dengan ahli mikologi sebelum mengonsumsi jamur liar. Keselamatan Anda adalah prioritas utama.
